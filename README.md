# 永齡希望小學雲科大分校 SEL 資源中心

社會情緒學習（SEL）課程工具包，供師生與家庭日常使用。

## 網站連結

部署後的入口頁：`https://yonglinytu.github.io/hopeshool/sel/`

---

## 檔案結構

放進 `hopeshool` repo 的 `sel/` 子資料夾：

```
hopeshool/
└── sel/
    ├── index.html                  ← 入口頁（SEL 資源中心）
    ├── hope-cards.html             ← HOPE 情緒六大家族牌卡
    ├── mindfulness-parents.html    ← 正念隨身行｜家長實踐指南
    ├── mindfulness-teacher.html    ← 正念隨身行｜師生課堂實踐手冊
    ├── dialogue-handbook.html      ← 親子覺察對話手冊
    └── README.md
```

---

## 四個資源簡介

| 資源 | 對象 | 核心功能 |
|---|---|---|
| HOPE 情緒六大家族牌卡 | 教師・家長・學生 | 84 張情緒牌卡，六大家族 |
| 正念隨身行（家長版） | 家長 | 情境推薦呼吸法、五感尋寶任務、計時練習 |
| 正念隨身行（教師版） | 教師 | 教案流程、教室情境快查、教師引導語 |
| 親子覺察對話手冊 | 家長 | 五個時間點 × 50 個對話題目 × 接話建議 |

---

## 部署步驟

`hopeshool` repo 的 GitHub Pages 已經開啟，只需要：

1. 進入 `hopeshool` repo
2. 點擊 **Add file → Upload files**
3. 建立路徑 `sel/`，將六個檔案全部上傳進去
4. Commit changes，等待約 1 分鐘後自動上線

> 不需要重新設定 Pages，沿用現有設定即可。

---

## 更新方式

直接在 GitHub 上傳新版本的同名 html 檔案即可覆蓋，Pages 會自動重新部署。
